# Decisions

## Accepted

- Nubel Store is the storefront brand.
- Customer account creation uses Google OAuth only.
- Admin dashboard access stays separate from customer authentication.
- 3D print quoting begins as an upload and preview workflow before marketplace matching.
- Project Brain Map is installed as a file-first project memory tool.

## Rejected

- Do not expose an obvious admin dashboard CTA to public customers.
- Do not commit real environment secrets.
